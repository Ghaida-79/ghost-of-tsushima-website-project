# Ghost of Tsushima Website

## About the Project
This is a web project inspired by the game **Ghost of Tsushima**.  
The project presents information and interactive content related to the game, including pages that allow user interaction and form submission.

The project was developed as an individual project using PHP and a local database.

---

## Live Demo
https://ghaida-79.github.io/ghost-of-tsushima-website-project/

Some features will not function when running the project directly from GitHub, as the project depends on PHP and a local database.

---

A demo video is included to demonstrate the **feedback page** and how the email message appears when it is successfully sent.

## Email Functionality (PHPMailer)
PHPMailer was used to implement the email feature within the project.

- SMTP was configured to send emails
- Email credentials were removed from the uploaded files
- Credentials must be added locally for security reasons
